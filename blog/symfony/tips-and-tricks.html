Именованные ресурсы

Именованные ресурсы
Бывают случаи, когда необходимо включить в HTML-разметку несколько ресурсов. Например, при использовании Bootstrap или JQuery.
Типичный подход, используемый в данном случае выглядит следующим образом:

{% raw %} 

{% javascripts
    '@AppBundle/Resources/public/js/jquery.js'
    '@AppBundle/Resources/public/js/jquery.ui.js'
    '@AppBundle/Resources/public/js/bootstrap.js'
    '@AppBundle/Resources/public/js/*' %}
        <script src="{{ asset_url }}"></script>
{% endjavascripts %}
{% endraw %}

Но гораздо удобнее именовать часто подключаемые ресурсы.
Это делается так:

{% raw %} 

# app/config/config.yml
assetic:
  assets:
    bootstrap_js: # название именованного ресурса
      inputs:
        - '@AppBundle/Resources/public/js/jquery.js'
        - '@AppBundle/Resources/public/js/jquery.ui.js'
        - '@AppBundle/Resources/public/js/bootstrap.js'
{% endraw %}

И теперь мы можем спокойно использовать наш именованный ресурс в шаблоне:

{% raw %} 

{% javascripts
    '@bootstrap_js'
    '@AppBundle/Resources/public/js/*' %}
    <script src="{{ asset_url }}"></script>
{% endjavascripts %}
{% endraw %}
